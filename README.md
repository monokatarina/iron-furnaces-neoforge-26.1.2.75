## 📖 English

### About
Iron Furnaces is a mod that adds multiple tiers of furnaces (Copper, Iron, Silver, Gold, Diamond, Emerald, Crystal, Obsidian, Netherite, Million, Allthemodium, Vibranium, Unobtainium), each with energy generation, upgrades, and automation features (auto‑input/output, factory mode, redstone control).

This repository contains a **manually updated version** for **NeoForge 26.1.2.75 (Minecraft 1.21.4)**. The original mod by pizzaatime and XenoMustache was not officially updated; all changes were made by hand to fix crashes, update rendering, and adapt to the new GUI system.

### Features
- ✅ All furnace tiers work – smelting, energy, generators.
- ✅ Upgrades (speed, fuel, generator, factory, smoking, blasting) fully functional.
- ✅ Auto‑input / auto‑output and redstone control.
- ✅ Configuration GUI with directional controls.
- ✅ Crafting recipes updated to 1.21.4 syntax.
- ✅ Item textures and most GUI elements fixed.

### Known Issues (GUI)
- Some interface elements may appear slightly misaligned or the background widget may not be perfectly positioned. However, **all mechanics work correctly**.
- The augment button and configuration panels are functional, but visual positioning might need fine‑tuning depending on screen resolution.

### Installation
1. Install NeoForge 26.1.2.75 for Minecraft 1.21.4.+
2. Download the latest `ironfurnaces-4.3.2.jar` from the **Releases** tab.
3. Place the jar file into your `mods` folder.
4. Launch the game.
